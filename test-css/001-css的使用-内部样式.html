安倍访华第二天 这两个重要表态信息量很大大

七年来，日本首相第一次正式访问中国，享受在中国检阅仪仗队、鸣放19响礼炮的高规格待遇。显然有点激动的安倍，出发前就忍不住在推特上感叹：已经有七年没正式访问中国了……10月26日，安倍访华第二天，高潮时刻来了：他在北京连会三名中国领导人。西装革履的安倍，一直系着一条红色领带。特意查了一下安倍过往的照片，他系红领带的时候还真不多。显然，安倍这次是花了点心思的。
